EYE: Autonomous Key-Logger Extension

Timeline Jun – Jul 2025
Platform Chrome

Overview

EYE is a Chrome extension built with Manifest V3 that demonstrates advanced browser automation and data capture capabilities. The extension showcases modern web extension architecture including content scripts, background services, and real-time DOM event handling.

Key Features

• Content scripts for real-time DOM manipulation and event interception

• Background service worker for persistent data collection

• Local storage for data persistence across browser sessions

• Dynamic popup UI with interactive data visualization

• Manifest V3 compliance for modern security standards

Technical Implementation

The extension uses Chrome's Manifest V3 API specifications for improved security and performance. Content scripts interact with web pages in real-time, capturing keyboard events with full context. Data is stored locally and managed through the service worker, ensuring efficient background operations without impacting browser performance.

What I Learned

This project deepened my understanding of browser security, cross-document communication, and event-driven architecture. Implementing Manifest V3 requirements highlighted the evolution of web extension security practices.
